# KvImage - Image viewer by VS & Qt & OpenCV

It's a simple image viewer based on VS2019(v142), OpenCV 3.1.0 and Qt5.12.9. You can open an image, then make some operation such as zooming, dragging, etc.

# Dependencies

- VS2019
- OpenCV 3.1.0
- Qt 5.12.9
- VS Qt Plugin 2.4.3
